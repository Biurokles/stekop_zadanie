# 3D Wykres instalacja
Na początek należy sciągnąć projekt
`git clone https://github.com/Biurokles/stekop_zadanie.git`

Do zarządzania pakietami, korzystam z npm, tak więc i do tego dam tutorial do odpalania.
Na początek instalujemy zależności:
`npm install"`

A następnie odpalamy aplikację
`npm run start`
